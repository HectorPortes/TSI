Desafio

Elabore uma página web que:
- Tenha um campo para o usuário digitar seu nome, sobrenome e data de nascimento;
- Faça a validação dos dados, removendo espaços em branco e transformando tudo em maiúsculo e verificando se a data de nascimento contém o mínimo de dígitos necessários;
- Ao passar o mouse sobre o botão verificar, mostre ao usuário os dados já validados;
- Ao clicar no botão enviar, o usuário deve receber uma mensagem de confirmação de que os dados foram enviados, mascarados a partir da terceira letra e com data e hora do envio.
